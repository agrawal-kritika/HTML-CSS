ALL ABOUT CSS

There are 3 ways to apply CSS to an HTML page:

  1. Inline CSS
  2. Internal CSS
  3. External CSS

  1. Inline CSS:

Inline styles are place directly inside an HTML element in the code using the "style" attribute.

  1. Internal CSS:

Internal styles are place inside the head section of a particular webpage via the "style" tag.

  1. External CSS:

An External style sheet is a separate page that is then linked to the web page.
